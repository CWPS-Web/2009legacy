馬偕故居--馬偕的小白宮

        由當時淡水最著名的洋樓匠師 ( 土水司 ) 洪仔泉所包，洪仔泉傳說曾參與英國領事官邸的建築，累積不少蓋「番仔樓」的經驗，成了淡水和大稻埕地區的西洋建築匠師 ( 日後偕醫館、牛津學堂、女學堂和不少教堂也都出自其手 ) 。
       這種殖民地式的 建築最適合熱帶地區。馬偕以哩岸砂岩為壁材，因砂岩有較佳之隔熱效果。屋架木材則是由廈門進口之福杉。屋頂是四坡落水有高低兩階 ( 迴廊屋架略低於室內 ) ，除了前後的老虎窗左右也留有大型氣窗，採閩南紅瓦 ( 俗稱台灣瓦 ) 。地板則抬高四呎作為隔潮層。面朝西南，三面設有白堊迴廊，牆面外緣敷以白灰，不僅較易反射嬌陽，也讓建築充滿南歐地中海式的浪漫情調。
        這種建築可稱為最理想之天然空調家屋，迴廊和其內的百葉窗隔絕了部分的陽光，白天悶熱的空氣可由通風的天花板上升，由屋頂所留的通氣窗排出，而夜間儲存在隔潮層內的地面冷空氣，則提供室內的涼爽。 另外，迴廊不僅可遮陽擋雨，三面迴廊正面朝向河口和觀音山也可取得怡人和悠閒的視野，為馬偕博士取得樂山樂水的家居生活。馬偕博士的宿舍正面有七個拱廊，正面入口前有九級精美觀音石台階，讓它在悠閒中帶點寬宏的氣勢。
        馬偕 博士自 1875 年搬入西側的「小白宮」為家，除兩次回加拿大之外，他 1878 年 05 月 27 日 在此結婚，長女媽連、長子偕叡廉也在此出生，他唯一身前得見的外孫柯設偕也在 1900 年出生於此，翌年 6 月 02 日他也在此臨終，結束傳奇性一生。
